Vector integral transformations

Divergence

Gauss's (divergence) theorem: $$∫↙\V\(\bo ∇⋅\bo A↖{→}\)dV=∮↙{S_c}\bo A↖{→}⋅d\bo s$$
Stokes's theorem: $$∫↙\S\(\bo ∇×\bo A↖{→})⋅ds=∮↙{L}\bo A↖{→}⋅d\bo l$$
Green's first theorem: $$∮↙{S}\(f\bo ∇g\)⋅ds=∫↙\V\bo ∇⋅\(f\bo ∇g\)dV$$
$$=∫↙\V\[f\bo ∇^2g+\(\bo ∇f\)⋅\(\bo ∇g\)\]dV$$
Green's second theorem: $$∮↙\S\[f\(\bo ∇g\)-g\(\bo ∇f\)\]⋅d\bo s$$
$$=∫↙\V\(f\bo ∇^2g-g\bo ∇^2f\)dV$$

where

$$\bo A↖{→}$$vector field
$$dV$$volume element
$$S_c$$closed surface
$$V$$volume enclosed
$$S$$surface
$$d\bo s$$surface element
$$L$$loop bounding $S$
$$d\bo l$$line element
$f$,$g$scalar fields
